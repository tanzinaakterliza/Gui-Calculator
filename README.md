# GUI Calculator

A simple GUI Calculator built using Java Swing that performs basic arithmetic operations through an interactive graphical user interface.

## Project Overview

This project demonstrates the use of Java Swing components, event handling, and object-oriented programming concepts in Java. Users can enter two numbers and perform different arithmetic operations by clicking the corresponding buttons.

## Features

- Addition of two numbers
- Subtraction of two numbers
- Multiplication of two numbers
- Division of two numbers
- Divide-by-zero error handling
- Invalid input validation
- User-friendly graphical interface
- Event-driven programming using ActionListener

## Technologies Used

- Java
- Java Swing
- AWT Event Handling
